A ransomware is a type of malware from cryptovirology that prevents users from accessing their systems or personal files. Furthermore the person behind the ransomware threatens to publish the victim's data unless a ransom is paid. That's the reason of the name, a combination of ransom and malware.
Some simple ransomware lock the system in a way which is not difficult to fix with a little bit of knowledge. There are however more advanced types of ransomwares which use a technique called cryptoviral extortion, which encrypts the files. Today's methods of encrypton take huge computational power to be deceived.


" In fact it is said that without the decryption key, recovering the files is an intractable problem, meaning that is a problem that can be solved in theory, but in practice it would take too many resources. "

Therefore what's left for the victim is to pay a ransom in order to get back the access to his/her files. Cryptocurrencies help this process, since they make the tracing of money very difficult if not impossible.

WannaCry

The WannaCry ransomware attac was a worldwide cyberattack which targeted computers running windows. It started on Friday 12 May 2017 and the starting point was Asia. Within a day, the code was reported to have infected more than 230'000 computers in over 150 countries. The computers affected by this attack were those which still had not installed the Microsoft security update from April 2017. Along with those, computer running an old version of Microsoft such as Microsoft XP were at risk since no security patches had been released for years. However the vast majority of the infected computers were running Windows 7.
In December 2017 it was formally asserted that North Korea was behind the attack. It is also known that the National Security Agency (NSA) had already discovered the vulnerability present in that Microsoft 7 version, but prefered to use it to create an exploit for its own offensive work, rather than report it to Microsoft.
