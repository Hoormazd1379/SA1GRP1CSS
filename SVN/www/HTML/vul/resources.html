Description

Uncontrolled resources consuption is a type of vulnerability that can occur if the size of Limited resources is not controlled.
What is limited resources?
Limited resource or System resource is device that is connected to a computer, wheres it is interal or external. It can be memory, CPU, Network connections and so on.
So, if the number of resources is not limited, then in any attack case in which this resources will be allocated, then the Server Resource manager can crush. COnsequently making software slower.

Consequences

Detection

Mitigation

In order to avoid problems such that one needs to put a limit to the amount of resources, which in turn will not let attacker to allocate uncontrolled resources. Consequently the software will be less vulnerable.
