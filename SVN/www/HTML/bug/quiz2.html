Question 2

What is a Severe Runtime Error in Videogame?

Severe is a game bug which restricts any movement from the player, without crashing or closing the game.

Question 3

When does a Runtime Error appear?

Runtime errors occur during execution of the software, so such errors occur during the interpretation phase.

Question 4

Why do problems related to time exist?

Problems relating to time exist because in an electronic device the space for keeping track of the time is finite.

Question 5

What can happen when in the code a bug is present?

In the worst case the program could crash, otherwise it will not behave like the programmer intended. In the second case, it may lead to some vulnerabilities.
