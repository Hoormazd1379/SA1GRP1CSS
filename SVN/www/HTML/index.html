Vulnerabilities

Vulnerability is a weakness of a system which can be exploited by a threat actor to perform unauthorized actions within the system. Some vulnerabilities are directly tied with bugs in the code, others are an innate repercussion of the type of the software, and others still are the result of a lack of implementation of mitigation strategies. What they all have in common is that they're not a security hazard by themselves, but only when a third party exploits them.

Bugs and Errors

A bug in informatics, is an error in a software that can cause several problems in the execution of the software itself. A software with bugs is defined buggy. There are different types of bugs that can cause different types of problems, which the severity may be high depending on quantity and severity of the bugs. Generally, a bug can be caused mainly by three circumstances: syntax error, logic error and runtime error.
